Advance Java
-------------
Why Advance Java ? 
Advance java is mainly used to create Web Application.

What is web application?

Application which runs on the browser and whose execution takes place on server is known as web application. It basically follows client server architecture. 
                request
       |----------------------->  
     client  <----HTTP---->  server
       <-----------------------|    
		response
